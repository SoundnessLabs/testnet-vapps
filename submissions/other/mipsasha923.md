### Description
RandomIdeaY is a tool for generating and sharing fun blockchain challenges verified via Soundness Layer zk-proofs. Users can submit solutions, compete on leaderboards, and share their results socially.

### SL Integration  
Soundness Layer will be used to verify submissions with zk-proofs, ensuring each challenge is unique and authentic.

## Technical

### Architecture
Frontend communicates with backend API, which verifies challenges on SL. Proofs are stored on-chain, metadata on IPFS.

### Stack
- **Frontend**: React
- **Backend**: Node.js
- **Blockchain**: Soundness Layer + Ethereum testnet
- **Storage**: IPFS

### Features
1. Generate and submit creative challenges
2. Verify submissions via SL zk-proofs
3. Leaderboard and social sharing

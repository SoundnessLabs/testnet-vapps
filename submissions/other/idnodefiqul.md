### Description
SafeVault is a decentralized file storage platform where users can upload their files to IPFS/WALRUS and earn rewards through token staking.  
The main problem addressed is the high cost and lack of security in traditional file hosting. With this vApp, data remains censorship-resistant, permanently accessible, and supported by a sustainable economic model.

### SL Integration  
SafeVault will integrate with Soundness Layer (SL) for:
- User identity verification when submitting files.  
- Secure validation of staking and reward transactions.  
- Connecting storage nodes with smart contracts for full transparency.  

## Technical

### Architecture
Frontend web app → API backend → smart contract on SL → decentralized storage (IPFS/WALRUS).  
- Metadata stored in database, files stored in IPFS.  
- Smart contract handles staking and rewards.  
- Backend Node.js acts as middleware between frontend and SL.  

### Stack
- **Frontend**: React + Tailwind  
- **Backend**: Node.js (Express)  
- **Blockchain**: Soundness Layer (SL), EVM compatible  
- **Storage**: IPFS + WALRUS  

### Features
1. Upload & download files via IPFS  
2. Token staking with reward distribution  
3. Dashboard to track storage usage & rewards

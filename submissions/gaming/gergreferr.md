### Description
RandomIdeaX is a fun tool to generate and share random creative challenges. Users can post prompts and others solve them with gamified feedback.

### SL Integration  
Soundness Layer will be used to verify user submissions with zk-proofs, ensuring uniqueness and authenticity of each challenge.

## Technical

### Architecture
Frontend communicates with backend API, which verifies challenges on SL. SL proofs are stored on-chain, while content metadata is kept in IPFS.

### Stack
- **Frontend**: React
- **Backend**: Node.js
- **Blockchain**: SL + Ethereum testnet
- **Storage**: IPFS

### Features
1. Generate and submit creative challenges
2. Verify challenges via SL zk-proofs
3. Leaderboard and social sharing

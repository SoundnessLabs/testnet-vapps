## Technical Idea

I want to build a simple Tic-Tac-Toe game where each win generates a zero-knowledge proof. That proof will be submitted to the Soundness Layer using the CLI. Once verified, the result updates a small scoreboard so players can see their confirmed wins. This shows how games can use proofs in a fun and practical way.

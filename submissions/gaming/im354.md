
# vApp Proposal: Soundy Crew WL Competition Game

## Category
gaming

## Project Overview
Soundy Crew WL Competition Game is a Scratch-based web game where players compete to earn whitelist spots for the Soundy Crew PFP NFT collection (supply: 111).  
Players only need to play the game, take a screenshot of their score, and wait for the contest results. No wallet connection is required.

## Technical Architecture with Soundness Layer
- **Frontend**: Scratch web game.  
- **Game Logic**: Collect points and avoid obstacles.  
- **Soundness Layer Usage**:  
  - Proof system used for fair contest verification (submissions checked through Soundness infrastructure).  
  - Ensures contest is transparent and tamper-proof.  

## Development Timeline

# Game Memory Flip - On-chain Mini Game

## Overview
This is a simple on-chain memory game built on top of Soundness.  
The idea is to let players test their memory skills by flipping hidden cards and trying to find matching pairs.  
Each action (flip / match) can be recorded on-chain, proving the fairness and uniqueness of every game session.

## Why Soundness?
Soundness layer allows verifiable proof of moves without relying on centralized servers.  
This ensures that the game is transparent, tamper-proof, and players can trust the results.

## Features
- Flip two cards to find a matching pair.
- Score is tracked and can be stored on-chain.
- Simple and lightweight, easy to extend into multiplayer.

# BOMB

## Overview
BOMB is a small browser game I made with plain HTML, CSS, and JavaScript.  
The idea is simple: you click on the board and try to avoid the hidden bombs.  
If you’re lucky (and a bit careful), you can clear the board without blowing up.

I wanted to keep it light, easy to run, and something quick to play.  
This is my first try at making a vApp for the Soundness testnet under the **Gaming** category.

## How it Works
- At the start, you pick how many bombs will be placed (3, 5, or 10).  
- Click on any tile to reveal it.  
  - If it’s safe, it shows a number telling you how many bombs are around.  
  - If it’s a bomb 💣 → game over.  
- The game ends when either you hit a bomb or you open all the safe spots.

## Why I Built This
I wanted to test something simple but fun for Soundness.  
Instead of overthinking it, I went with a “guess the bomb” type of game — it’s quick, replayable, and works fine without any extra libraries.  
It’s also a good way to show interactivity on the Soundness layer without making things too complicated.

## Run the Game
1. Download or clone the repo.  
2. Open the file `index.html` in your browser.  
3. That’s it. No build steps, no installs.

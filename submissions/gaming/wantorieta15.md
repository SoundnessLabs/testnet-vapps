Description

GameVerse Arena is a Web3 battle arena where players can fight each other using unique NFT characters.  
Problem solved: In traditional games, asset ownership is usually centralized. With this vApp, assets are truly owned by players, can be traded, and match results are verified on-chain to prevent cheating.


SL Integration

The game leverages Soundness Layer for:

- **Player Identity** → unique player verification & anti-bot protection  
- **Asset Ownership** → character & item ownership secured by SL  
- **Match Verification** → battle results verified & recorded on-chain  
- **Fair Play** → anti-cheat mechanisms powered by SL proofs  


Technical

**Architecture**

- Game client (Unity WebGL) connected to a Node.js backend  
- Smart contracts on Soundness Layer for items, match records, and leaderboard  
- IPFS for character metadata storage  

**Stack**

- Frontend/Game Engine: Unity WebGL + React dashboard  
- Backend: Node.js + GraphQL API  
- Blockchain: Soundness Layer + EVM-compatible chain  
- Storage: WALRUS/IPFS for asset metadata  


Features

1. PvP Battle (real-time arena fights)  
2. NFT-based characters & weapons  
3. Leaderboard & player ranking  
4. Marketplace for asset trading  
5. SL-powered verifiable match records

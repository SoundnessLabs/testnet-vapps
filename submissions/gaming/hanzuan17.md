# vApp Proposal: Sudoku Gaming dApp

- Category: gaming  
- Proposer: hanzuan17  
- Discord ID: 1177885253575131229  

## Short Description
I propose adding a new community-based game, Sudoku.  
This game provides entertainment while also offering logical challenges that are simple yet engaging.  

## Technical Architecture
- The Sudoku game will run as a vApp on top of the Soundness Layer.  
- Integration with the Soundness Layer will enable:  
  - Cryptographic validation of game results/scores.  
  - Fairness assurance between players (anti-cheating).  
- A simple web-based UI/UX will be developed for easy community access.  

## Development Timeline
- Week 1: Project setup & initial integration with Soundness Layer.  
- Week 2: Implementation of Sudoku mechanics (board generation, validation).  
- Week 3: Score integration with Soundness Layer.  
- Week 4: Community testing & feedback collection.  

## Additional Notes
The goal of this proposal is to enrich the gaming variety within the Soundness ecosystem through a simple yet valuable community-driven game

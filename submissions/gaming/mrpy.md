### Description
Soundness Puzzle is a browser-native fan-made puzzle game built with HTML, CSS, and JavaScript.
It focuses on simplicity, instant playability, and visual polish. Players can rearrange puzzle tiles to complete an image, all directly in the browser without installation, wallet connection, or sign-up.

The goal is to create a fun, minimal, and stylish experience that showcases how casual games can be both lightweight and visually engaging while remaining accessible to anyone.

### SL Integration

## Project Overview

**Title:** GameChain Arena  
**Category:** Gaming

**Summary:**  
GameChain Arena is a decentralized Web3 gaming platform that allows players to compete in skill-based games with on-chain score tracking, player rewards, and NFT-based assets. It leverages smart contracts to ensure fair competition and uses zk-proofs to verify player actions without revealing sensitive data.

**Goals:**
- Create a fun and provably fair P2E game
- Integrate on-chain identity to prevent multi-accounting
- Use NFTs for in-game assets and progress tracking

**Tech Stack:**
- Solidity / Hardhat
- React / Next.js
- Ethers.js
- IPFS

## Project

- **Category**: identity

### Description
Solves the problem of lacking a universal and secure way to verify digital identity in Web3. Allows users to create a verified decentralized profile usable across multiple dApps. Minimizes repeated KYC checks, reduces the risk of forged data, and simplifies access to DeFi and social services.

### SL Integration
Uses Soundness Layer for authenticity verification through zk-validators, secure storage and management of identity proofs, cross-dApp integration via SL API for seamless login and trustless verification.

## Technical

### Architecture
SL smart contracts handle identity verification, while zk-proofs protect privacy. Decentralized profiles are stored off-chain, with frontend for users and backend for blockchain/SL logic.

### Stack
- **Frontend**: React + Next.js
- **Backend**: Node.js
- **Blockchain**: SL + Ethereum
- **Storage**: WALRUS + IPFS

### Features
1. Create and manage decentralized profiles.
2. zk-proof identity verification without exposing personal data.
3. Unified login integration for external dApps.

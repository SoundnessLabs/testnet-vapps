 ## Overview
 DecentraID is a decentralized identity application that allows users to verify identity on top of the Soundness Layer without having to trust a centralized third party.

 ## Key Features
 - Self-Sovereign Identity: users control their identity data.
 - ZK-Based Verification: uses zero-knowledge proof for privacy.
 - Soundness Layer Integration: leveraging contracts on the Soundness testnet.

 ## Technical Architecture
 - Frontend: React + Wagmi
 - Smart Contracts: Solidity
 - ZK Proof: zk-SNARK

 ## Development Timeline
 - Week 1-2: Basic architecture &amp; smart contracts
 - Week 3-4: Frontend &amp; integration
 - Week 5: Testing &amp; bug fixing
 - Week 6: Deploy &amp; documentation

 ## Why Soundness Layer?
 Soundness Layer provides a distributed security &amp; verification mechanism for identity applications without relying on a centralized server.

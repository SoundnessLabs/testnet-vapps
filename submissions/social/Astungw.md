### Description
EpicWorld is a decentralized social hub designed to connect communities, enable knowledge sharing, and reward meaningful contributions with on-chain incentives.
It solves the problem of centralized platforms controlling user content by ensuring data ownership, censorship-resistance, and transparent on-chain reputation.

### SL Integration  
* Uses Soundness Layer for identity verification without traditional KYC.
* Integrates zkProofs for private yet verifiable reputation and account validation.
* Enables low-fee micropayments on Soundness Layer for tipping, donations, and community rewards.

## Technical

### Architecture
* Client (Web & Mobile) → User interface for communities and content interaction.
* Backend API → Manages user accounts, content feeds, and rewards logic.
* Soundness Layer → Provides zkProof identity, security, and micropayments.

### Stack
- **Frontend**: React + Tailwind
- **Backend**: Node.js (Express)
- **Blockchain**: Soundness Layer + EVM compatible
- **Storage**: IPFS for decentralized data storage

### Features
1. Community creation & content feed
2. On-chain tipping & reward system
3. zk-based reputation system

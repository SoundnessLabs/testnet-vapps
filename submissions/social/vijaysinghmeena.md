## Short Summary
Build a Telegram-based bilingual quiz bot (Hindi + English) that uses an Excel-backed question bank, per-question timers, and a scoreboard. Integrate with Soundness Layer to record verified contest results and optionally reward winners.

## Motivation / Why
This increases community engagement and demonstrates Soundness Layer usage for light-weight dApps (vApps) in social contexts.

## Technical Architecture
- **Frontend:** Telegram Bot (Bot API)  
- **Backend:** Python/Node.js server for quiz logic  
- **Soundness Layer Integration:** Submit quiz results to Soundness testnet for verification  
- **Database:** Excel/JSON for question storage  

## Development Timeline
- Week 1: Setup Telegram bot + Soundness CLI integration  
- Week 2: Implement quiz flow & scoring  
- Week 3: Add leaderboard + rewards  
- Week 4: Testing & deployment

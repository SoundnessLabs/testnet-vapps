### Description
ConnectSphere is a decentralized social networking platform that empowers users to control their data and monetize their content without intermediaries. It solves the problem of centralized platforms exploiting user data by leveraging blockchain for transparent data ownership and rewarding creators with tokens for engagement.

### SL Integration  
ConnectSphere will use Soundness Layer (SL) for secure, scalable, and low-cost transactions to manage user interactions and rewards. Specific SL features include:
SL's transaction layer: For processing microtransactions for content monetization.
Smart contract execution: To enforce transparent reward distribution based on user engagement metrics.
Identity verification: Using SL's decentralized identity tools to ensure secure and private user authentication.

## Technical

### Architecture
ConnectSphere uses a modular architecture with a frontend for user interaction, a backend for off-chain processing, and SL for on-chain operations. Users interact via a web app, with data stored on decentralized storage and transactions validated on SL. The system ensures scalability through SL's high-throughput transaction processing

### Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express  
- **Blockchain**: Soundness Layer (SL) for transactions and smart contracts
- **Storage**: IPFS for user-generated content, WALRUS for metadata storage

### Features
1. Decentralized user profiles with full data ownership
2. Token-based rewards for content creation and engagement  
3. Encrypted private messaging with SL-based identity verification

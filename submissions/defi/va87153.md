Abstract

zkPay is a DeFi payment application that enables users to send funds between wallets anonymously and verifiably using zero-knowledge proofs on the Soundness Layer. The goal is to create a private, efficient, and fraud-resistant transaction system in the Web3 ecosystem.

Project Description

Current DeFi platforms expose transaction data publicly, leading to privacy issues and traceability risks. zkPay solves this by using zkApps on the Soundness Layer to prove wallet ownership and balance without revealing sensitive data.

Key features:

Users can register zkPay accounts.

Send and receive testnet payments.

Prove ownership of wallet and minimum balance privately.

Use verified ZK identity badges when sending large transactions.

Soundness Layer Integration

zkPay integrates with the Soundness Layer in the following ways:

Users generate zk-proofs for wallet ownership and available balance.

The zkApp on the Soundness Layer verifies these proofs.

The backend uses soundness-cli to interact with the zkApp and validate user proofs.

A dummy smart contract (on testnet) executes the actual fund transfers.

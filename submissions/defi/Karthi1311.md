### Description
A decentralized app that enables **recurring subscription payments** in crypto.  
Users can subscribe to services (content platforms, SaaS, or communities) by authorizing a recurring stablecoin payment directly from their wallet.  
This solves the lack of a seamless way for merchants to accept subscriptions in crypto.

### SL Integration  
- Use **Soundness Layer (SL)** for fast, low-cost recurring transactions.  
- Leverage SL’s **secure settlement layer** to ensure reliability of recurring payments.  
- Smart contracts on SL manage subscription schedules and merchant payouts.

## Technical

### Architecture
- User connects wallet → approves subscription → smart contract automates recurring payments → merchant wallet receives funds.  
- On-chain logic handles subscription cycles, while simple UI allows merchants to define plans and users to manage subscriptions.

### Stack
- **Frontend**: React with tailwind (basic subscription shadcn UI) 
- **Backend**: Node.js (payment orchestration) 
- **Blockchain**: Soundness Layer (primary)  
- **Storage**: IPFS/WALRUS for plan metadata  

### Features
1. Set up subscriptions with one-click approval.  
2. Merchants define subscription plans (price, duration).  
3. Users can pause or cancel anytime.  
4. On-chain history of all payments.

## Innovation
Recurring payments are a missing piece in Web3.  
This vApp introduces a **global, decentralized subscription system** that works like credit card auto-pay, but with stablecoins and no middleman.  
It brings **mainstream subscription models** (Netflix, Spotify, SaaS) into the crypto world.

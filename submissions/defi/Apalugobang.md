### Description
What problem does your vApp solve? What does it do?

DecentraSwap is a decentralized exchange (DEX) built on the Soundness Layer (SL) to provide cross-chain swaps with additional ZK-proof security.
Key issues addressed:

Liquidity fragmentation across chains.

Security of cross-chain transactions by leveraging cryptographic verification from SL.

Simplified user experience for bridging and swapping.

### SL Integration  
How will you use Soundness Layer? What specific SL features?

ZK Verification: Utilizes SL as cross-chain transaction verification (proof relay).

Identity Layer: Uses SL wallet signatures for user authentication.

Scalability: Utilizes SL batching to reduce cross-chain gas fees.

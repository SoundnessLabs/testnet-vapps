# Idea Title: Proof Tracker

## Description
Proof Tracker aims to enhance CLI usability by tracking proof submissions in real-time and providing detailed logs for each submission.

## Key Features
1. Real-time monitoring of proof generation progress.
2. Detailed logs for each submission including timestamps and status.
3. Alerts for failed proofs or retries needed.

## Initial Implementation
- Python file: proof_tracker.py
- CLI command: soundness-cli --track
- This PR adds only the idea file and basic structure; full implementation will follow.

## Benefits
- Improves transparency of proof submission process.
- Helps users quickly identify and fix failed submissions.
- Enhances user experience for active CLI users.
